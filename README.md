# Clinic Management COP4870
By Isaiah Rohrer

This is a clinic management system that allows users to manage patients, physicians, and appointments.

## Features
- Add, edit, and delete patients
- Add, edit, and delete physicians
- Add, edit, and delete appointments
- View patient history
- View physician schedule
- View appointment details
- Add treatments to appointments


## To Run
run ./make.sh in project root
